# office-scripts/

This folder holds Office.js / Excel automation scripts that sit alongside the
Python pipeline but are not called by it.

## phase1-analysis-sheet.js

Exploratory Office.js script that builds the initial analysis sheet layout
inside a SignalStack workbook. Kept as reference — not part of the current
pipeline flow. You can run it from the Excel "Automate" tab against a copy
of one of the `SignalStack_*.xlsx` templates if you want to see the
sheet-construction logic.
